# 籃球數據紀錄器 #

## 發想起源 ##

* 提供校園系隊將紙本記錄方式，轉為電子。
* 利於統計分析數據
* 傳統紙本記錄表 VS 手機APP記錄程式



## 製作方法 ##

* 使用工具： Swift、SQLite、XCode
* 將使用者輸入資料儲存於SQLite，並於查詢時將資料呼叫出來
